# Cloudflare Audit Logs MCP

Official Cloudflare MCP for accessing account audit logs.

## Features

- **User Activity**: Track all user actions and changes
- **API Monitoring**: Monitor API access and usage
- **Authentication Events**: Login attempts, MFA changes
- **Permission Tracking**: Role and access changes
- **Configuration Audit**: Zone and domain modifications
- **Security Policies**: Firewall and security updates
- **Compliance Reports**: SOC 2, ISO 27001 support
- **Alerting**: Notifications for suspicious activities
- **SIEM Integration**: Export logs for external analysis

## Authentication

Requires a Cloudflare API token with Audit Logs read permissions.
