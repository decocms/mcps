# Teamwork MCP

## Project Description

**Teamwork MCP** is a Model Context Protocol (MCP) server that integrates with Teamwork.com for project management, task tracking, and team collaboration.

### Purpose

This MCP server allows client applications to:
- Create, update, and manage projects, tasks, and milestones
- Track time, budgets, and resource allocation across projects
- Collaborate with team members through messages and file sharing

### Key Features

- 📁 **Project Management**: Create and manage projects with tasks, lists, and milestones
- ✅ **Task Tracking**: Assign, prioritize, and update task status and due dates
- ⏱️ **Time Logging**: Record and retrieve time entries for billing and productivity
- 👥 **Team Collaboration**: Access messages, comments, and team member assignments
- 📊 **Reporting**: Retrieve project progress, budgets, and workload reports

## Authentication

Authentication is handled via OAuth through the MCP connection at `https://mcp.ai.teamwork.com`. Users authorize access through Teamwork's OAuth flow.
