# ThoughtSpot MCP

## Project Description

**ThoughtSpot MCP** is a Model Context Protocol (MCP) server that gives AI assistants access to ThoughtSpot's AI-powered analytics platform for searching data, generating insights, and querying business intelligence content.

### Purpose
- Search and query business data using ThoughtSpot's natural language and AI-powered search
- Retrieve answers, liveboards, and data visualizations from your ThoughtSpot instance
- Integrate AI-driven analytics into developer and agent workflows

### Key Features
- 🔍 Run natural language searches against your connected data sources
- 📊 Retrieve answers and AI-generated insights from ThoughtSpot
- 📋 Access liveboards, pinboards, and saved analyses
- 🧠 Leverage SpotIQ for automated anomaly detection and trend analysis
- 🔗 Query metadata and data source definitions across your analytics environment

## Authentication

Authentication is handled via OAuth through the MCP connection. Connect using the URL `https://agent.thoughtspot.app/mcp` and authorize with your ThoughtSpot account credentials.
