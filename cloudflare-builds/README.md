# Cloudflare Builds MCP

Official Cloudflare MCP for managing Pages builds and deployments.

## Features

- **Build Management**: Trigger and monitor build processes
- **Deployment Control**: Manage production and preview deployments
- **Build Logs**: Access detailed build logs and error messages
- **Configuration**: Manage build settings and environment variables
- **Git Integration**: Automatic deployments from Git repositories
- **Framework Support**: Works with all major static site frameworks

## Authentication

Requires a Cloudflare API token with Pages permissions.
