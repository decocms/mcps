# Cloudflare DNS Analytics MCP

Official Cloudflare MCP for DNS query analysis and monitoring.

## Features

- **Query Monitoring**: Track DNS query volumes and trends
- **Record Type Analysis**: Breakdown by A, AAAA, MX, TXT, etc.
- **Response Codes**: Monitor NXDOMAIN, SERVFAIL, and errors
- **Geographic Distribution**: Query origins worldwide
- **Performance Metrics**: Response times and latency
- **Top Domains**: Most queried domains and subdomains
- **Attack Detection**: Identify DNS DDoS patterns
- **DNSSEC Analytics**: Validation rates and issues

## Authentication

Requires a Cloudflare API token with DNS Analytics permissions.
