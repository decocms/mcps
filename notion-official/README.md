# Notion MCP

Official Notion Model Context Protocol (MCP) server for managing Notion workspaces, pages, and databases.

## Features

- **Page Management**: Create, read, update, and organize Notion pages
- **Block Operations**: Manage rich content blocks (text, headings, lists, images, etc.)
- **Database Integration**: Full CRUD operations on Notion databases
- **Query & Filter**: Advanced database queries with filters and sorts
- **Collaboration**: Workspace sharing and team collaboration
- **Real-time Sync**: Keep content synchronized across your applications

## Authentication

Requires Notion OAuth authentication or integration token. Create an integration at:
https://www.notion.so/my-integrations
