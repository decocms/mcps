# Atlassian Rovo MCP

## Project Description

**Atlassian Rovo MCP** is a Model Context Protocol (MCP) server that integrates AI assistants with Atlassian Rovo, providing access to Jira project management and Confluence knowledge base content.

### Purpose
- Query and manage Jira issues, projects, sprints, and workflows from AI tools
- Search and read Confluence pages, spaces, and documentation
- Streamline engineering and team workflows by connecting AI assistants to your Atlassian ecosystem

### Key Features
- 🎯 Create, update, and search Jira issues and epics
- 🔄 Manage sprint planning, backlogs, and workflow transitions
- 📖 Search and read Confluence pages and spaces
- 👥 Query team members, assignments, and project roles
- 🤖 Leverage Rovo AI agents for intelligent knowledge retrieval

## Authentication

Authentication is handled via OAuth through the MCP connection. Connect using the URL `https://mcp.atlassian.com/v1/mcp` and authorize with your Atlassian account.
