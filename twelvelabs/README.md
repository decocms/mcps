# TwelveLabs MCP

## Project Description

**TwelveLabs MCP** is a Model Context Protocol (MCP) server that integrates with TwelveLabs for video indexing, semantic search, analysis, and embedding generation.

### Purpose

This MCP server allows client applications to:

- Index and search video content using natural language queries
- Generate embeddings and analyze video at a semantic level
- Extract insights, summaries, and structured data from video files

### Key Features

- 🎬 **Video Indexing**: Index video content for fast semantic and multimodal search
- 🔍 **Semantic Search**: Find moments in videos using natural language descriptions
- 🧠 **Video Understanding**: Generate summaries, chapters, and key highlights
- 📐 **Embedding Generation**: Create vector embeddings for video content and clips
- 🏷️ **Content Analysis**: Detect objects, actions, text, and speech within videos

## Authentication

Authentication is handled via OAuth through the MCP connection at `https://mcp.twelvelabs.io`. Users authorize access through TwelveLabs' OAuth flow.
