# Postman MCP

Official Postman Model Context Protocol (MCP) server for managing Postman collections, environments, and API workflows.

## Features

- **Collection Management**: Create, read, update, and organize Postman collections
- **Request Operations**: Manage API requests, headers, and parameters
- **Environment Variables**: Configure and manage environment variables
- **Workspace Collaboration**: Share and collaborate on API documentation
- **Test Automation**: Execute API tests programmatically
- **API Monitoring**: Monitor API performance and health

## Authentication

Requires a Postman API key. You can generate one at:
https://go.postman.co/settings/me/api-keys
