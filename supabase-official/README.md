# Supabase MCP

Official Supabase Model Context Protocol (MCP) server for managing Postgres databases and backend services.

## Features

- **SQL Execution**: Run SQL queries and manage Postgres databases
- **Migration Management**: Apply and track database migrations with version control
- **Edge Functions**: Deploy and manage Deno-based serverless functions
- **Development Branches**: Create isolated environments for safe testing
- **TypeScript Types**: Generate type-safe interfaces from database schemas
- **Performance Monitoring**: Get security and performance recommendations
- **Log Analysis**: Access logs across all Supabase services
- **Database Extensions**: Manage Postgres extensions and features
- **Authentication**: Configure auth services and policies
- **Storage**: Manage file storage configurations

## Authentication

Requires a Supabase Access Token. Generate one at:
https://supabase.com/dashboard/account/tokens
