# Blockscout MCP

## Project Description

**Blockscout MCP** is a Model Context Protocol (MCP) server that gives AI assistants access to Blockscout's blockchain explorer for querying on-chain data, transactions, addresses, and smart contracts.

### Purpose
- Query blockchain data including transactions, blocks, and address balances
- Look up smart contract details, ABIs, and verified source code
- Retrieve token information and on-chain analytics across supported networks

### Key Features
- 🔗 Look up transactions, blocks, and on-chain events
- 🏦 Query address balances, token holdings, and activity history
- 📜 Retrieve verified smart contract source code and ABIs
- 🪙 Search and inspect ERC-20, ERC-721, and ERC-1155 tokens
- 📊 Access on-chain analytics and network statistics

## Authentication

No authentication is required for public data. Connect using the URL `https://mcp.blockscout.com/mcp` to access the Blockscout blockchain explorer API.
