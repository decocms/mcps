# Jotform MCP

## Project Description

**Jotform MCP** is a Model Context Protocol (MCP) server that connects AI assistants to Jotform for managing forms, viewing submissions, and automating form-based workflows.

### Purpose
- Access and manage Jotform forms and their settings from AI-powered tools
- Retrieve and analyze form submission data programmatically
- Automate form creation and notification workflows

### Key Features
- 📝 List, create, and update forms and form fields
- 📥 Retrieve and filter form submissions
- 📊 Access submission analytics and response summaries
- 🔔 Manage form notifications and autoresponders
- 🔗 Configure webhooks and third-party integrations

## Authentication

Authentication is handled via OAuth through the MCP connection. Connect using the URL `https://mcp.jotform.com/` and authorize with your Jotform account.
