# Cloudflare GraphQL MCP

Official Cloudflare MCP for GraphQL Analytics API.

## Features

- **Custom Queries**: Build flexible analytics queries
- **Multi-dimensional**: Aggregate by time, location, device, etc.
- **Multiple Datasets**: HTTP, firewall, DNS, Workers, and more
- **Complex Filters**: Precise data filtering and time ranges
- **Data Combining**: Join multiple data sources
- **Historical Analysis**: Access historical trends
- **Performance**: Optimized field-level selection
- **Integration**: Connect with BI tools and dashboards

## Use Cases

- Custom analytics dashboards
- Executive reporting
- Performance analysis
- BI tool integration
- Automated monitoring

## Authentication

Requires a Cloudflare API token with Analytics read permissions.
