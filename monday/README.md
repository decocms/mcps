# Monday.com MCP

## Project Description

**Monday.com MCP** is a Model Context Protocol (MCP) server that connects AI assistants to your monday.com workspace for managing projects, boards, items, and automations.

### Purpose
- Interact with monday.com boards and items directly from AI-powered tools
- Automate project management tasks such as creating, updating, and tracking work items
- Query and manage automations, columns, and workflows across your workspace

### Key Features
- 📋 Read and write monday.com boards, groups, and items
- 🔍 Search and filter items across your workspace
- ⚙️ Manage column values, statuses, and assignments
- 🤖 Trigger and configure board automations
- 📊 Retrieve workspace and user information

## Authentication

Authentication is handled via OAuth through the MCP connection. Connect using the URL `https://mcp.monday.com/mcp` and authorize with your monday.com account credentials.
