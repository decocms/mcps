# Cloudflare AI Gateway MCP

Official Cloudflare MCP for managing AI API requests.

## Features

- **Multi-Provider Support**: OpenAI, Anthropic, Google, Azure, and more
- **Response Caching**: Reduce costs and improve latency
- **Rate Limiting**: Control request quotas per user/app
- **Usage Analytics**: Monitor AI API consumption
- **Fallback Logic**: Automatic failover across providers
- **Cost Control**: Spending limits and alerts
- **Access Control**: Authentication and authorization
- **Data Privacy**: Anonymize sensitive information

## Authentication

Requires a Cloudflare API token with AI Gateway permissions.
