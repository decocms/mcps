# Cloudflare Radar MCP

Official Cloudflare MCP for accessing global internet trends and insights.

## Features

- **Traffic Analysis**: Global internet traffic patterns and trends
- **Security Intelligence**: Real-time DDoS and threat monitoring
- **Technology Adoption**: Track HTTP/3, IPv6, DNSSEC usage
- **Network Performance**: Internet quality metrics by region
- **Routing Data**: BGP announcements and routing information
- **Outage Monitoring**: Track internet disruptions worldwide
- **Bot Detection**: Automated traffic pattern analysis
- **Domain Rankings**: Popular domains and trends

## Authentication

Cloudflare API token (optional for some endpoints, required for advanced features).
