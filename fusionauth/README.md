# FusionAuth MCP

## Project Description

**FusionAuth MCP** is a Model Context Protocol (MCP) server that provides access to FusionAuth's documentation for identity, authentication, and authorization solutions.

### Purpose

This MCP server allows client applications to:

- Query FusionAuth documentation for implementation guidance
- Find code examples and configuration references for auth workflows
- Access API documentation for users, applications, and tokens

### Key Features

- 📚 **Documentation Search**: Search FusionAuth's full documentation library
- 🔐 **Auth Workflow Guidance**: Get implementation help for OAuth, SAML, OIDC, and MFA
- 💻 **Code Examples**: Retrieve language-specific SDK and API usage examples
- 🔑 **API Reference**: Access detailed API documentation for FusionAuth endpoints
- 🏗️ **Configuration Guides**: Find guidance on tenant, application, and connector setup

## Authentication

This MCP server provides access to public FusionAuth documentation via `https://ai.fusionauth.dev/mcp/docs`. No authentication is required to query the documentation.
