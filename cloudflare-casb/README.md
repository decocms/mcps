# Cloudflare CASB MCP

Official Cloudflare MCP for Cloud Access Security Broker functionality.

## Features

- **Security Scanning**: Detect misconfigurations in SaaS apps
- **Data Exposure**: Identify sensitive data at risk
- **Shadow IT**: Monitor unauthorized cloud app usage
- **Compliance Monitoring**: GDPR, HIPAA, SOC 2 checks
- **Access Auditing**: Track user permissions and changes
- **Integration Control**: Monitor third-party OAuth grants
- **DLP Policies**: Data loss prevention enforcement
- **Security Reports**: Posture assessments and compliance
- **Alerting**: High-risk activity notifications

## Supported Applications

Google Workspace, Microsoft 365, Salesforce, Slack, GitHub, Zoom, and more.

## Authentication

Requires a Cloudflare API token with CASB permissions and connected SaaS applications.
