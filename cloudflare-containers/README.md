# Cloudflare Containers MCP

Official Cloudflare MCP for managing containerized applications on the edge.

## Features

- **Container Deployment**: Deploy Docker containers globally
- **Automatic Scaling**: Scale based on demand automatically
- **Load Balancing**: Distribute traffic across container instances
- **Health Monitoring**: Track container health and performance
- **Environment Management**: Configure variables and secrets
- **CI/CD Integration**: Automated deployment pipelines
- **Global Distribution**: Run containers close to your users

## Authentication

Requires a Cloudflare API token with container management permissions.
