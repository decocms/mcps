# Todoist MCP

## Project Description

**Todoist MCP** is the official Model Context Protocol (MCP) server for Todoist, enabling AI assistants to manage tasks, projects, and workflows.

### Purpose

This MCP server allows client applications to:

- Create, update, and complete tasks in Todoist
- Manage projects, labels, and priorities
- Automate task management workflows with AI

### Key Features

- ✅ **Task Management**: Create, update, complete, and organize tasks
- 📁 **Project Organization**: Manage projects and sections
- 🏷️ **Labels & Priorities**: Set priorities, labels, and due dates
- 🔄 **Workflow Automation**: Automate recurring task patterns
- 🤖 **AI Integration**: Natural language task management through AI assistants

## Authentication

Authentication is handled via OAuth through the MCP connection:

```
URL: https://ai.todoist.net/mcp
```
