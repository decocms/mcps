# Medusa MCP

## Project Description

**Medusa MCP** is a Model Context Protocol (MCP) server that provides AI assistants with access to the MedusaJS headless commerce documentation and developer resources.

### Purpose

- Browse and search the official MedusaJS documentation from within AI-powered tools
- Quickly retrieve guides, API references, and integration documentation
- Accelerate development by surfacing relevant Medusa concepts and code examples on demand

### Key Features

- 📚 Full access to MedusaJS official documentation content
- 🔍 Search across guides, API references, and tutorials
- 🛒 Coverage of commerce concepts: products, orders, carts, customers, and more
- 🔌 Plugin and module integration documentation
- ⚡ Fast lookup of configuration and deployment guides

## Authentication

No authentication is required. Connect using the URL `https://docs.medusajs.com/mcp` — the server provides public read-only access to documentation.
