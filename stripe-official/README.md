# Stripe MCP

Official Stripe Model Context Protocol (MCP) server for managing payments, subscriptions, and billing operations.

## Features

- **Customer Management**: Create and manage customer profiles, payment methods
- **Payment Processing**: Process one-time and recurring payments securely
- **Subscription Billing**: Manage plans, subscriptions, and usage-based billing
- **Invoice Operations**: Generate, send, and track invoices automatically
- **Refunds & Disputes**: Handle refunds and dispute management
- **Financial Reporting**: Access detailed analytics and reporting
- **Multi-currency Support**: Process payments in multiple currencies
- **Fraud Prevention**: Built-in fraud detection and prevention

## Authentication

Requires a Stripe API key (secret key). Get your keys at:
https://dashboard.stripe.com/apikeys

⚠️ **Security**: Never commit your secret keys. Use environment variables.
