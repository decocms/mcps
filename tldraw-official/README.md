# tldraw MCP Server Official

This is the **official tldraw MCP Server**, provided directly by the tldraw team for AI-powered visual collaboration.

## About tldraw

tldraw is a collaborative whiteboard and diagramming tool. With this official MCP, AI agents can:

- **Create shapes** - Add new visual elements to the canvas
- **Edit shapes** - Modify existing canvas elements
- **Delete shapes** - Remove shapes from the diagram

## Connection

This MCP connects to the official tldraw server at:

```
https://tldraw-mcp-app.tldraw.workers.dev/mcp
```

## How to Use

1. Install this MCP through the registry
2. Start drawing and diagramming with AI assistance
