# Cloudflare AutoRAG MCP

Official Cloudflare MCP for Retrieval-Augmented Generation applications.

## Features

- **Automatic Embeddings**: Generate vectors from documents automatically
- **Vector Database**: Semantic search and similarity matching
- **LLM Integration**: Connect with Large Language Models
- **Chunking Strategies**: Optimize document splitting for retrieval
- **Hybrid Search**: Combine vector and keyword search
- **Knowledge Management**: Version and update document bases
- **Performance Optimization**: Caching and indexing
- **Accuracy Monitoring**: Track RAG pipeline performance

## Authentication

Requires a Cloudflare API token with AI and Vectorize permissions.
