# Cloudflare Logs MCP

Official Cloudflare MCP for accessing and analyzing log data.

## Features

- **HTTP Request Logs**: Detailed request/response data
- **Firewall Events**: Security actions and blocked requests
- **DNS Query Logs**: DNS resolution data
- **Worker Logs**: Execution logs and errors
- **Cache Analytics**: CDN performance metrics
- **Bot Detection**: Challenge and bot traffic logs
- **Logpush**: Export logs to external services
- **Custom Filters**: Query specific log subsets

## Authentication

Requires a Cloudflare API token with Logs permissions.
