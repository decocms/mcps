# Redpanda MCP

## Project Description

**Redpanda MCP** is a Model Context Protocol (MCP) server that provides access to Redpanda's documentation for Kafka-compatible streaming data infrastructure.

### Purpose

This MCP server allows client applications to:

- Query Redpanda documentation for configuration and deployment guidance
- Find API references and code examples for producers, consumers, and streams
- Access guidance on Redpanda cluster management and operations

### Key Features

- 📚 **Documentation Search**: Search Redpanda's full documentation for concepts and guides
- ⚡ **Streaming Reference**: Access Kafka-compatible API docs for producers and consumers
- 🏗️ **Deployment Guides**: Find configuration and deployment instructions for self-hosted and cloud
- 🔧 **Operations Support**: Get guidance on cluster tuning, monitoring, and maintenance
- 🔄 **Schema Registry**: Access documentation for schema management and data contracts

## Authentication

This MCP server provides access to public Redpanda documentation via `https://docs.redpanda.com/mcp`. No authentication is required to query the documentation.
