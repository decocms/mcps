# Cloudflare DEX MCP

Official Cloudflare MCP for Digital Employee Experience monitoring.

## Features

- **Endpoint Monitoring**: Track device connectivity and performance
- **Application Performance**: Monitor app load times and responsiveness
- **Zero Trust Metrics**: Network access and security compliance
- **VPN/Tunnel Quality**: Connection stability and performance
- **Network Analysis**: WiFi and connectivity issue detection
- **Device Compatibility**: Track OS and device problems
- **Security Compliance**: Policy enforcement monitoring
- **Alerting**: Notifications for connectivity issues
- **Experience Reports**: Overall digital experience metrics

## Authentication

Requires a Cloudflare API token with DEX/Zero Trust permissions.
