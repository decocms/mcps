# Cloudflare Browser MCP

Official Cloudflare MCP for browser automation and rendering.

## Features

- **Browser Automation**: Run Puppeteer and Playwright scripts
- **Web Scraping**: Extract data from dynamic websites
- **Automated Testing**: Cross-browser and device testing
- **Screenshot & PDF**: Capture page visuals and generate PDFs
- **Monitoring**: Track website availability and changes
- **Global Access**: Run browsers from edge locations worldwide
- **Scale**: Automatic load balancing and resource management

## Authentication

Requires a Cloudflare API token with Workers Browser Rendering permissions.
