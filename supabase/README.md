# Supabase Official MCP

Binding for the official Supabase MCP.

## MCP URL

```
https://mcp.supabase.com/mcp?project_ref=ovbxhfblvsuilhnoievu
```

## About

This is a binding for the official Supabase MCP, which provides direct integration with your Supabase project, including:

- SQL query execution
- Table management
- CRUD operations
- And more native Supabase features

## Note

The `project_ref` in the URL is project-specific. To use with another project, replace `ovbxhfblvsuilhnoievu` with your own project_ref.
